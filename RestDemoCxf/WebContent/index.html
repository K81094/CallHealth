REST Service Working fine


This is a REST webservice made of Apache CXF and Spring Framework


How to test the service

0. You can request User details without inserting anything but you will get response as following

URL: POST : http://localhost:8080/RestDemoCxf/services/rest/UserManager/fetchAllUsers/

JSON Req : body : {}

Response : { users: null errorMessage: null success: true }
1. First you need to insert a user value into the databse
You can do that by following ways: Open POSTMan and pass this query with JSON POST request

URL : POST: http://localhost:8080/RestDemoCxf/services/rest/UserManager/insertUser

Json POST :body: {"user":{"name":"Rocky","email":"someting@nospam.org","birthDate":"1975-01-01","city":"Omaha","state":"NE"}}

Response : { "users": null, "errorMessage": null, "success": true }
2. You can test the service by fetching user from the service

URL : POST: http://localhost:8080/RestDemoCxf/services/rest/UserManager/fetchAllUsers/

Json Post : body : {}

Response : { "users": [ { "id": 0, "name": "Rocky", "email": "someting@nospam.org", "birthDate": 157766400000, "city": "Omaha", "state": "NE" }, { "id": 1, "name": "Rocky", "email": "someting@nospam.org", "birthDate": 157766400000, "city": "Omaha", "state": "NE" } ], "errorMessage": null, "success": true }
